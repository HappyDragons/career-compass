# 求职罗盘 —— 大学生求职导航工具

> 从迷茫到清晰，一站式解决大学生求职困惑

一款专为大学生设计的求职辅助工具，覆盖求职全流程：看懂JD → 评估技能 → 规划成长 → 准备面试 → 优化简历。

## 功能模块

### 1. JD翻译器
- 选择岗位或粘贴招聘描述，翻译成大白话
- 提取关键技能词，标注核心能力优先级
- 判断是否应届友好，提供薪资参考

### 2. 技能雷达
- Canvas雷达图可视化展示技能差距
- 滑块调节自评分数，实时对比岗位要求
- 红/黄/绿三色反馈，一眼看出薄弱项

### 3. 成长路线
- 从大一到大四的四阶段成长时间线
- 每个阶段配备具体可执行的任务清单
- 覆盖8大热门岗位方向

### 4. 模拟面试
- 岗位定制化面试题库
- 手风琴展开答题思路提示
- AI反馈：检查回答的STAR结构、数据支撑、篇幅

### 5. 简历优化
- STAR法则引导式输入
- 一键生成3种专业简历表述
- 支持一键复制到剪贴板

## 覆盖岗位

| 方向 | 岗位 |
|------|------|
| 技术研发 | 前端开发工程师、后端开发工程师、数据分析师 |
| 产品运营 | 产品经理、运营专员 |
| 设计创意 | UI/UX设计师 |
| 商业职能 | 市场营销、人力资源 |

├── css/
│   └── style.css       # 样式表
├── js/
│   ├── data.js         # 岗位数据（8个岗位完整信息）

## 技术实现

- **纯前端**：HTML + CSS + 原生 JavaScript，零依赖
- **Canvas 雷达图**：程序化绘制，支持实时更新
- **响应式设计**：适配桌面、平板和手机
- **Google Fonts**：Noto Sans SC

## 使用方法

直接在浏览器中打开 `index.html` 即可使用。

```bash
# 本地预览
python3 -m http.server 8000
# 访问 http://localhost:8000
```

## 公益赛道说明

本项目为 TRAE SOLO 挑战赛公益赛道作品，选题方向：**大学生求职迷茫**。

针对大学生求职过程中的核心痛点：
- 看不懂招聘JD的专业术语
- 不知道自己的技能差距有多大
- 缺少清晰的成长路径规划
- 面试准备无从下手
- 简历描述不够专业

通过5个工具模块，提供从认知到行动的完整解决方案。
