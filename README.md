# 求职罗盘 —— 大学生求职导航工具

> 从迷茫到清晰，一站式解决大学生求职困惑

一款专为大学生设计的**全栈**求职辅助工具，前后端完整实现，覆盖求职全流程：看懂JD → 评估技能 → 规划成长 → 准备面试 → 优化简历。

## 技术架构

- **后端**：Node.js + Express
- **数据库**：SQLite（better-sqlite3）
- **认证**：JWT（jsonwebtoken）+ bcryptjs 密码加密
- **前端**：原生 HTML + CSS + JavaScript
- **可视化**：Canvas 雷达图
- **字体**：Google Fonts (Noto Sans SC)

## 功能模块

### 用户系统
- 注册/登录（JWT Token 认证）
- 个人档案管理（学校、专业、年级、目标岗位）
- 登录日志记录

### 数据看板
- 技能评估次数、面试练习题数、简历生成次数、探索岗位数
- 最近动态时间线
- 技能评估概览（各岗位最新评分）

### 1. JD翻译器
- 选择岗位或粘贴招聘描述，翻译成大白话
- 提取关键技能词（40+关键词库），标注核心能力优先级
- 判断是否应届友好，提供薪资参考

### 2. 技能雷达
- Canvas雷达图可视化展示技能差距
- 滑块调节自评分数，实时对比岗位要求
- 红/黄/绿三色反馈，一眼看出薄弱项
- **登录后可保存评估记录，下次自动加载**

### 3. 成长路线
- 从大一到大四的四阶段成长时间线
- 每个阶段配备具体可执行的任务清单
- 覆盖8大热门岗位方向

### 4. 模拟面试
- 岗位定制化面试题库（每岗位4题）
- 手风琴展开答题思路提示
- 即时反馈：检查STAR结构、数据支撑、篇幅，给出评分
- **登录后自动保存练习记录和得分**

### 5. 简历优化
- STAR法则引导式输入（情境/任务/行动/结果）
- 一键生成3种专业简历表述
- 支持复制到剪贴板
- **登录后可保存简历记录**

### 历史记录
- 技能评估、面试练习、简历记录三类历史数据
- 支持查看和删除

## 覆盖岗位

| 方向 | 岗位 |
|------|------|
| 技术研发 | 前端开发工程师、后端开发工程师、数据分析师 |
| 产品运营 | 产品经理、运营专员 |
| 设计创意 | UI/UX设计师 |
| 商业职能 | 市场营销、人力资源 |

## 数据库设计

| 表名 | 说明 |
|------|------|
| users | 用户账号（用户名、邮箱、密码哈希） |
| profiles | 个人档案（目标岗位、学校、专业、年级） |
| skill_assessments | 技能评估记录（岗位、各技能评分、差距摘要） |
| interview_records | 面试练习记录（题目、回答、反馈、评分） |
| resumes | 简历生成记录（STAR四要素 + 生成文本） |
| login_logs | 登录日志（IP、User-Agent） |

## API 接口

| 方法 | 路径 | 说明 | 认证 |
|------|------|------|------|
| POST | /api/auth/register | 注册 | - |
| POST | /api/auth/login | 登录 | - |
| GET | /api/auth/me | 当前用户信息 | 需要 |
| GET | /api/profile | 获取档案 | 需要 |
| PUT | /api/profile | 更新档案 | 需要 |
| POST | /api/skill | 保存技能评估 | 需要 |
| GET | /api/skill | 获取评估列表 | 需要 |
| GET | /api/skill/latest/:jobId | 获取最近评估 | 需要 |
| DELETE | /api/skill/:id | 删除评估 | 需要 |
| POST | /api/interview | 保存面试记录 | 需要 |
| GET | /api/interview | 获取面试记录 | 需要 |
| POST | /api/resume | 保存简历 | 需要 |
| GET | /api/resume | 获取简历列表 | 需要 |
| DELETE | /api/resume/:id | 删除简历 | 需要 |
| GET | /api/stats | 数据看板统计 | 需要 |

## 使用方法

```bash
# 安装依赖
npm install

# 启动服务器（默认端口3000）
npm start

# 访问 http://localhost:3000
```

首次启动会自动创建 SQLite 数据库文件。

## 公益赛道说明

本项目为 TRAE SOLO 挑战赛公益赛道作品，选题方向：**大学生求职迷茫**。

针对大学生求职过程中的核心痛点：
- 看不懂招聘JD的专业术语
- 不知道自己的技能差距有多大
- 缺少清晰的成长路径规划
- 面试准备无从下手
- 简历描述不够专业

通过用户系统 + 5个工具模块 + 数据看板，提供从认知到行动的完整解决方案，并持久化用户学习进度。
